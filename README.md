Portfolio Website
=================

A new simpler rewrite of the portfolio website.

Install
=======
An installer script will be added later when things are finalized. For now:

    npm install -g angular-cli

OS X only:

    brew install watchman

Development
===========
To run use:

    ng serve --port 8000

Testing
=======
Unit tests:

    ng test

E2Es using Protractor:

    ng e2e
